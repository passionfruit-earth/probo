# Compliance Input Materials

Source materials used for ISO 27001 compliance work.

## Structure

```
input/
├── questionnaires/     # Security questionnaire responses
└── interviews/         # Interview notes (if any)
```

## Questionnaires

Place completed questionnaires here. These provide context for:
- Organization details (team, access, systems)
- Current security practices
- Policy decisions

## Usage

When working on compliance tasks, reference these files for accurate org information instead of asking again.
