## Common GraphQL Queries

### Get all measures
```graphql
{
  node(id: "2delmZUAAAEAAAAAAZx_7gpa_vVWYEaD") {
    ... on Organization {
      measures {
        edges {
          node {
            id
            name
            state
          }
        }
      }
    }
  }
}
```

### Create a measure
```graphql
mutation CreateMeasure($input: CreateMeasureInput!) {
  createMeasure(input: $input) {
    measureEdge {
      node {
        id
        name
      }
    }
  }
}
```

Variables:
```json
{
  "input": {
    "organizationId": "2delmZUAAAEAAAAAAZx_7gpa_vVWYEaD",
    "name": "Measure name",
    "description": "Description",
    "category": "Policy|Process|Technical|People"
  }
}
```

### Link measure to control
```graphql
mutation {
  createControlMeasureMapping(input: {
    controlId: "<control_id>",
    measureId: "<measure_id>"
  }) {
    controlEdge {
      node {
        id
      }
    }
  }
}
```

### Update measure state
```graphql
mutation {
  updateMeasure(input: {
    id: "<measure_id>",
    state: IMPLEMENTED
  }) {
    measure {
      id
      state
    }
  }
}
```

**Valid states:** `NOT_STARTED`, `IN_PROGRESS`, `NOT_APPLICABLE`, `IMPLEMENTED`
